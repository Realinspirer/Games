Hyper-Casual Runner

This is a simple, and fun 2D side-scrolling based game, that contains many cool challenges and obstacles for the player to avoid.

Several Power-ups

This game contains many fun power-ups that you can buy and even get during the run! They all will help you in their unique ways!

In-game Shop

This game has collectibles that you can collect while in the run. You can buy power-ups and upgrade in the in-game shop using the collected packages!

Upgrading System

In the shop, you can upgrade your power-ups to make them better during the run!
